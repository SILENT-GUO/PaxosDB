## How to run a sample program
1. Go to ```main.cpp``` in ```/application``` Folder
2. Run
```shell
    ./main <thisip:thisport> otherip1:otherport1,otherip2:otherport2
```
For example,
```shell
    ./main localhost:1111 localhost:2222,localhost:3333
```
